# Automated Customer Churn System

## Project Overview
The Automated Customer Churn System is a data analytics project developed to analyze customer churn behavior using Python, MySQL, and Power BI. The project helps identify customer retention patterns and business insights through interactive dashboards and automated data processing workflows.

---

## Technologies Used
- Python
- Pandas
- MySQL
- Power BI
- SQL

---

## Features
- Data Cleaning using Python
- Automated Data Processing
- MySQL Database Integration
- Customer Churn Analysis
- Revenue Analysis
- Risk Category Analysis
- Interactive Power BI Dashboard
- KPI Visualization

---

## Project Workflow
Raw Data (Excel/CSV)
→ Python Data Cleaning
→ Processed Data
→ MySQL Database
→ Power BI Dashboard

---

## Dashboard KPIs
- Total Customers
- Total Revenue
- Churn Customers
- Average Monthly Charges
